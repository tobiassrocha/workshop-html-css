Camadas de desenvolvimento

Existem 3 linguagens básicas que utilizamos para criar websites:

  • HTML: é a linguagem que irá exibir a informação.
  • CSS: é a linguagem que vai deixar essa informação estilizada.
  • Javascript: é a linguagem que vai manipular comportamentos na página.

HTML

O acrônico HTML significa em inglês: HyperText Markup Language.

Se você tiver que guardar alguma coisa sobre o que é HTML, guarde isso: HTML serve para dar significado e organizar a informação dos websites.

E como ele faz isso?

É simples, através das tags.

Tags são o conjunto de caracteres que formam um elemento, ou seja, quando nos referenciamos à Tag "p" estamos falando disso: <p> isso é um parágrafo <p/>

Tipos de Tags

Existem 2 tipos de tags:

  • Tags que necessitam de fechamento. 
    A tag do elemento de parágrafo necessita de fechamento.
  • Tags que não necessitam de fechamento. 
    Texto utilizando 
     quebra de linha

Elementos

Elementos são formados a partir de tags e entre as tags é que está o conteúdo do elemento.

Exemplo


							

								Isto é um link

Não confunda Tags com Elementos

As tags servem para marcar onde começam e terminam os elementos.

Já os elementos são uma parte conceitual/semântica que têm um começo e fim determinados.

Atributos

Atributos são informações que passamos na tag para que ela se comporte da maneira esperada. Os atributos também possuem nome e um valor.

  • Atributos globais: class, id, style.
  • Atributos específicos: src, draggable, target.

Metatags

Representada pela tag <meta> as metatags são diferenciadas das demais por não possuírem nenhum efeito visual na página, e sim por serem responsáveis por ações externas, como por exemplo informar à buscadores informações à respeito da página.

Tipos de metatags

  • Charset: indica o formato de codificação de caracteres.
  • Description: breve descrição da página.
  • Author: nome do autor da página.
  • Keywords: metatag responsável por fornecer informações relevantes aos motores de busca.
  • Viewport: metatag útil para aplicações responsivas.

Estrutura básica de um documento html

  • Definição do documento (doctype)
  • Cabeçalho (head)
  • Corpo (body)

CSS

O Cascading Style Sheets (CSS) é uma linguagem utilizada para definir a apresentação (aparência) de documentos que adotam para o seu desenvolvimento linguagens de marcação (como XML, HTML e XHTML e etc..).

Expectativa

Realidade

Por que o CSS foi criado?

Com a evolução dos recursos de programação, as tecnologias estavam adotando cada vez mais estilos e variações para deixá-las mais elegantes e atrativas para os usuários.

O css foi desenvolvido para habilitar a separação do conteúdo e formato de um documento de sua apresentação, incluindo elementos como cores, formatos de fontes e layout.

Classes

As classes são uma forma de identificar um grupo de elementos. Através delas, pode-se atribuir formatação a VÁRIOS elementos de uma vez.


						.classe1 {
							background: blue;
						}

Id

Os ids são uma forma de identificar um elemento, e devem ser ÚNICOS para cada elemento. É como se fossem impressões digitais de nossos dedos ou RGs. Através deles, pode-se atribuir formatação à um elemento em especial. 


						#idUm {
							background: blue;
						}

O que são pré processadores CSS

São programas que permitem gerar código CSS a partir de uma sintaxe exclusiva do pré-processador.

Pontos positivos no uso de pré processadores

  • Possibilita a criação de variáveis, mixins, operações matemáticas e até extends
  • Facilitam a manutenção do código
  • Auxilia o trabalho com componentes
  • Acaba com problemas comuns do dia-a-dia, como ficar escrevendo os prefixos dos navegadores
